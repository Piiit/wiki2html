<h2>What is that?</h2>
This is a <b>wiki</b> parser basically, where you can easily produce HTML-compliant files in a go.
<h2>Some features</h2>
<ul>
<li>implements wiki syntax (<a href='http://en.wikipedia.org/wiki/Creole_%28markup%29'>creole wiki markup</a>)</li>
<li>produces HTML compliant output</li>
<li>easy to use</li>
<li>free and portable</li>
</ul>

<h2>More information</h2>
You can find more information about the syntax <a href='http://www.wikicreole.org/wiki/AllMarkup'>here</a>
